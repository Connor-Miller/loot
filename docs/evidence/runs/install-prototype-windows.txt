===== FINDINGS =====
- WORKS: detect, download, unpack, ~/.loot/bin placement, HKCU PATH prepend
  (persisted, fresh-shell resolvable), idempotent re-run (single PATH entry),
  no-admin. Ship the generated installer unmodified.
- GAP 1: `loot` has no `--version`/`version` command. Release blocker.
- GAP 2: loot-cli-installer.ps1 has NO checksum verification (loot-cli-installer.sh
  DOES). Windows-first integrity asymmetry — needs a decision.
- CONFIG: install-path set to ~/.loot/bin (not dist default CARGO_HOME).
